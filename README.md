# Walmart-Sales-Analytics-Using-SQL
This project uses advanced SQL to analyze Walmart sales data and generate business insights. It covers sales performance, customer behavior, category trends, profitability, demand patterns, and branch level efficiency using CTEs, window functions, aggregations, ranking, time based analysis, and stored procedures.

#Project Summary
Performed comprehensive retail sales analysis by first cleaning and preprocessing raw transactional data using Python, followed by advanced SQL analytics to evaluate sales performance, customer behavior, pricing patterns, and growth trends. Combined Python based data cleaning with SQL techniques such as CTEs, window functions, time series analysis, and stored procedures to deliver accurate and actionable business insights.


#Features
#-Identifies highest rated product categories per branch
#-Finds busiest sales days for each branch
#-Analyzes top revenue generating categories by city
#-Calculates month over month sales growth
#-Detects low performing categories using sales decline and rating thresholds
#-Builds a profitability index using revenue, profit margin, and ratings
#-Identifies repeat purchase behavior using time window analysis
#-Detects price and quantity outliers by category
#-Calculates customer demand elasticity based on price and quantity changes
#-Compares weekday and weekend sales performance
#-Finds cities outperforming company average growth
#-Detects branches with consistent sales decline
#-Identifies peak sales hours based on revenue
